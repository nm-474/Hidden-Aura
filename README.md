# Hidden Aura

**Hidden Aura** is a modern, responsive web application designed with a focus on minimalist aesthetics and seamless user experience. Hosted on Netlify, this project showcases clean design principles and high-performance front-end architecture. Hide and reveal secret messages in images using AES encryption and steganography — all in your browser!

🔗 **Live Demo:** [https://hiddenaura.netlify.app/](https://hiddenaura.netlify.app/)

---

## ✨ Features

- **Modern UI/UX:** A sleek, "aura-inspired" design with smooth transitions and professional typography.
- **Fully Responsive:** Optimized for all screen sizes, from mobile devices to large desktop monitors.
- **Fast Performance:** Light-weight assets and optimized code for lightning-fast loading speeds via Netlify.
- **Interactive Elements:** Engaging user interface components built with modern web standards.

## 🛠️ Built With

- **HTML5 & CSS3:** For structural semantic markup and advanced styling (including Flexbox/Grid).
- **JavaScript (ES6+):** For dynamic interactions and logic.
- **Netlify:** For continuous deployment and global hosting.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
- A modern web browser.
- A local server (like Live Server for VS Code) or Node.js installed (if using build tools).

### Installation
1. Clone the repository:
   ```bash
   git clone [https://github.com/your-username/hidden-aura.git](https://github.com/your-username/hidden-aura.git)

## Steps
1. Open `index.html`.
2. Upload an image, type your secret, and set a password.
3. Click “Hide Message” to generate a new image.
4. Download and later upload the stego image with the correct password to reveal the message.
